
Release steps:

1. Test all examples to the latest released version
2. mvn release:clean
3. mvn release:prepare
4. mvn release:perform
5. Follow steps to release:
    https://docs.sonatype.org/display/Repository/Sonatype+OSS+Maven+Repository+Usage+Guide#SonatypeOSSMavenRepositoryUsageGuide-8.ReleaseIt
6. Update and test all examples to the latest released version
